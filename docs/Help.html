Introduction

This application was created to help manage flight plans and view on a map. It is not meant to create/edit plans as there are many good applications for that. You can import flight plans, rate them, add some comments, etc.

The following data will be imported and saved. Fields in bold are editable.
  • Plan Name - Title from the XML
  • Plan Type (IFR or VFR)
  • Import Date
  • Departure Id
  • Destination ID
  • Rating - User assigned rating for this plan
  • Distance - Total distance calculated for the plan
  • Group the plan was flown with
  • Original file name - This is the file that was imported
  • Notes - Initially set to a count of the number of waypoints and airports, can be used for anything
  • File Create Date - The data the the plan file was created
  • Author - The author of the flight plan
  • Airport Count - The count of airports in the flight plan
  • Waypoint Count - The count of waypoints in the flight plan

Import Plan

Plans need to be imported into the application. Either select Import from the file menu or right click on the screen and select import. From the import screen, select one or more plan files to be imported. All plans will be imported with default fields saved.

Edit Plan

The editable fields can be updated directly in the table. Any edits are immediatly saved. Note that these updates are in an internal database and does not in any way affect the original plan file.

Sorting

The table is sortable by clicking any column headers. A second click will reverse the order of the sort.

Export Plan

The original plan is saved in the application. Right clicking on the table will bring up a menu with an Export option. This allows you to export the plan back to the drive for use later. It can be useful if the original plan was deleted.

Map Plan

Plans can be viewed on a map. Right clicking on the table will bring up a menu with a Map option. This allows you to view the plan on a map. The map can be zoomed in or out with the mouse.

Delete Plan

Right clicking on the table will bring up a menu with a Delete option. Delete will remove the plan from the application. It will not affect the plan saved on your harddrive.

Options

Settings

  • Overwrite existing plan - If this option is selected, when importing an existing plan (same filename and title) the plan will be overwritten with the new data. Only the non-editable fields will be replaced, user data, such as ratings, will not be chanegd

Select Columns

This page gives the ability to select which columns will be shown on the main page and which order they will be shown in
